USER INPUT & BRANCHING - Chore App

A program that can be used to decide who should do chores around the house.






Only two users are allowed to play at a time.

User is prompted to enter the chore and each user's name.

The Java Math random number is utilized within a method name decider to generate a number between 0 & 1

If the generated number is less than 0.5, user 1 gets the honor of doing the chore, other wise user 2 gets the honor of doing the chore.

The decider method should take the chore and each user's name as parameters.
